# echo-environment

## 環境変数
| 環境変数名   | default value | description                             |
|---------|---------------|-----------------------------------------|
| APP_ENV | `dev`         | `local`、`dev` など環境にあったものを想定しています。       |
| PORT    | `8080`        | listenするポートを指定してください。defaultは `8080`です。 |


```
go mod init github.com/jyonleon7/echo-environment
```

```
go mod tidy
```
